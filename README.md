# govconnect

This should be considered a work in progress.  We welcome pull requests and discussion via [github issues](https://github.com/18F/govconnect/issues).

# Requirements

- Ruby (2.2.3 recommended)
- Bundler gem

# Usage

- `bundle install`
- `jekyll serve`
- `open http://127.0.0.1/govconnect`
